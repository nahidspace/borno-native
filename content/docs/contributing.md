The full development guide lives in [`CONTRIBUTING.md`](https://github.com/nahidspace/borno-native/blob/master/CONTRIBUTING.md).

Before opening a change:

- Add a focused regression input for conversion changes.
- Run the console harness against `tools/oracle/wordlist.txt`.
- Keep oracle code and data development-only.
- Update attribution when adding third-party material.
- Test TSF changes in a fresh Notepad process.

Original project code and documentation are Apache-2.0. The bundled word-frequency resource and development oracle have separate terms; see `NOTICE`.
