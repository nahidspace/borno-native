## The path from key to script

1. Windows sends a key event to the TSF key event sink.
2. Borno Native keeps the current ASCII phonetic buffer.
3. `Convert()` tokenizes the buffer with longest-match rules.
4. Contextual conjunct pairs and vowel signs are emitted as Unicode.
5. A synchronous edit session replaces only the active composition.
6. Uniscribe shapes the result in the host application and in the suggestion popup.

The core converter is independently implemented. The development oracle in `tools/oracle/` is used for behavioral comparison only; its implementation and data are not shipped.

## Composition behavior

Space, punctuation, and Enter commit the current word. Backspace removes one raw phonetic key and re-converts the shortened buffer. Modifier keys such as Shift stay out of the buffer so uppercase phonetic forms like `brO` remain one composition.

## Suggestions

Suggestions are generated by re-converting alternate phonetic spellings, which keeps vowel-sign placement contextual. The bundled frequency resource ranks candidates, while the resource's separate CC BY-SA 4.0 terms are documented in `Avro.TSF/Resources/ATTRIBUTION.md`.
