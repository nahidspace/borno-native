# bn_wordfreq.tsv attribution

Bangla word-frequency data (top 25,000 words, Zipf frequency scale),
extracted via [wordfreq](https://github.com/rspeer/wordfreq) (Apache-2.0
licensed code) by Robyn Speer.

wordfreq's Bangla data is compiled from Wikipedia, OpenSubtitles, news text,
general web text, and Twitter, and is licensed **CC BY-SA 4.0**. Per that
license: this extracted list is a derivative of wordfreq's data and is
itself offered under CC BY-SA 4.0 -- attribution to wordfreq/rspeer and the
underlying corpora, share-alike for any further derivative of this specific
word list.

This applies to `bn_wordfreq.tsv` only, not to the rest of Borno Native's
source code, which is released under Apache-2.0.
